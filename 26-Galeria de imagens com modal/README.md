Galeria de imagens com visualização em modal desenvolvida em JavaScript com manipulação de DOM. A aplicação exibe um conjunto de imagens em formato de grade e permite ao usuário clicar em qualquer uma delas para visualizá-la em tamanho ampliado.

O sistema renderiza dinamicamente as imagens a partir de um array de URLs, criando elementos HTML e adicionando eventos de clique para cada item. Ao selecionar uma imagem, um modal é exibido com a versão ampliada. O modal pode ser fechado, retornando à visualização da galeria.

## MVP

- Lista de imagens pré-definidas
- Renderização dinâmica da galeria
- Evento de clique em cada imagem
- Abertura de modal com imagem ampliada
- Atualização do conteúdo do modal dinamicamente
- Fechamento do modal
