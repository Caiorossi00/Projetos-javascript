Gerador de números aleatórios desenvolvido em JavaScript com manipulação de DOM. A aplicação permite que o usuário informe a quantidade de dígitos desejada e, a partir desse valor, o sistema gera um número aleatório composto por algarismos de 0 a 9.

O número é construído dinamicamente através de um laço de repetição que seleciona índices aleatórios de um array de dígitos. Após a geração, o resultado é exibido diretamente na interface sem necessidade de recarregar a página.

## MVP

- Entrada para definir a quantidade de dígitos
- Validação para impedir valores inválidos
- Geração aleatória de dígitos de 0 a 9
- Construção dinâmica do número gerado
- Exibição do resultado na interface
