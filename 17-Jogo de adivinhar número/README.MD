Jogo do Número Mágico desenvolvido em JavaScript com manipulação de DOM. A aplicação permite que o jogador tente adivinhar um número aleatório gerado pelo sistema dentro de um intervalo definido (1 a 9).

O sistema gera um número mágico aleatório a cada jogada, recebe o palpite do jogador, valida se o valor está dentro do intervalo permitido e compara com o número mágico. Após a verificação, exibe na interface se o jogador venceu ou perdeu, mostrando o número correto quando o palpite não corresponde.

## MVP

- Entrada de palpite pelo jogador
- Geração aleatória do número mágico entre 1 e 9
- Validação do palpite para garantir que esteja dentro do intervalo
- Comparação entre palpite e número mágico
- Exibição de mensagem de vitória ou derrota
- Atualização dinâmica do resultado na interface
