Sistema de ranking de jogadores desenvolvido em JavaScript com manipulação de DOM. A aplicação permite adicionar jogadores com suas respectivas pontuações e exibir uma lista ordenada do maior para o menor desempenho.

Os dados são armazenados em um array de objetos, contendo nome e pontuação. A cada atualização, o sistema ordena os jogadores com base na pontuação e renderiza a lista na interface, exibindo a posição de cada participante no ranking.

## MVP

- Lista inicial de jogadores com pontuação
- Adição de novos jogadores
- Validação de entrada dos dados
- Ordenação do ranking por pontuação (decrescente)
- Exibição da posição no ranking
- Atualização dinâmica da lista na interface
- Limpeza dos campos após inserção
