Simulador de caixa eletrônico desenvolvido em JavaScript com manipulação de DOM. A aplicação permite realizar saques e depósitos, atualizando o saldo do usuário em tempo real na interface.

Ao realizar um saque, o sistema calcula automaticamente a menor quantidade possível de cédulas para compor o valor solicitado, utilizando uma estratégia gulosa baseada nas notas disponíveis. Após a operação, o saldo é atualizado e a distribuição das notas é exibida na tela.

O projeto tem como objetivo praticar lógica de programação, estruturas de repetição, arrays, objetos, manipulação de eventos e atualização dinâmica de elementos HTML, além de aplicar conceitos básicos de algoritmos para otimização de problemas.

## MVP

- Saque de valores com validação de saldo
- Depósito de valores com validação de entrada
- Cálculo automático da menor quantidade de notas para o saque
- Exibição da distribuição de cédulas na tela
- Atualização dinâmica do saldo após cada operação
- Interface interativa baseada em eventos de clique
