Temporizador regressivo desenvolvido em JavaScript com manipulação de DOM. O usuário pode definir um tempo inicial e iniciar a contagem regressiva através de um botão. O valor do timer é atualizado automaticamente na interface a cada segundo.

O sistema utiliza `setInterval` para controlar a contagem do tempo e interrompe automaticamente quando o valor chega a zero, exibindo uma notificação ao usuário. Também é possível reiniciar o temporizador, restaurando o valor inicial e interrompendo qualquer contagem em andamento.

## MVP

- Definição de tempo inicial pelo usuário
- Início da contagem regressiva através de botão
- Atualização do tempo na interface a cada segundo
- Interrupção automática quando o tempo chega a zero
- Alerta ao finalizar o tempo
- Função para reiniciar o temporizador
