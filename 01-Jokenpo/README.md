Um jogo interativo de Pedra, Papel e Tesoura desenvolvido em JavaScript puro com manipulação de DOM. O jogador escolhe uma das opções e o computador gera uma jogada aleatória automaticamente. O sistema compara as jogadas, determina o vencedor da rodada e atualiza o placar em tempo real.

A aplicação utiliza lógica condicional para implementar as regras do jogo, geração de números aleatórios para a decisão da CPU e eventos de clique para permitir a interação do usuário com a interface. O placar registra vitórias do jogador, vitórias do computador e empates, podendo ser reiniciado sem necessidade de recarregar a página.

O objetivo do projeto é praticar fundamentos essenciais de JavaScript, incluindo funções, estruturas condicionais, arrays, objetos, eventos e atualização dinâmica de elementos HTML.

## MVP

- Jogador vs computador
- Sistema de pontuação acumulativa
- Exibição do vencedor de cada rodada
- Atualização dinâmica do placar na tela
- Geração aleatória da jogada do computador
- Reinício do placar sem recarregar a página
