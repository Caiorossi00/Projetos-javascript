Simulador de rolagem de dado desenvolvido em JavaScript com manipulação de DOM. A aplicação permite que o usuário informe quantas vezes deseja rolar o dado e, a partir disso, gera resultados aleatórios para cada lançamento.

O sistema utiliza um array representando as faces do dado e um laço de repetição para executar múltiplas rolagens. Em cada iteração, um valor aleatório é selecionado e acumulado em uma string que representa o resultado final. Após o processamento, todos os valores sorteados são exibidos dinamicamente na interface.

## MVP

- Entrada para definir a quantidade de rolagens
- Geração aleatória de valores entre 1 e 6
- Execução de múltiplas rolagens em sequência
- Construção dinâmica do resultado final
- Exibição dos resultados na interface
