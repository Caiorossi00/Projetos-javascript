To-do list simples desenvolvida em JavaScript puro com manipulação de DOM. A aplicação permite adicionar e remover tarefas dinamicamente através de uma interface básica composta por um campo de texto, um botão de adição e uma lista exibida na página.

Cada tarefa inserida recebe um identificador único, permitindo que ela seja removida individualmente. O sistema valida a entrada do usuário para impedir a criação de tarefas vazias e atualiza a interface sempre que uma tarefa é adicionada ou excluída.

## MVP

- Adicionar novas tarefas
- Validação para impedir tarefas vazias
- Listagem dinâmica das tarefas na tela
- Exclusão individual de tarefas
- Atualização automática da interface após cada ação
