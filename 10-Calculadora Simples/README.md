Calculadora básica desenvolvida em JavaScript com manipulação de DOM. A aplicação permite realizar operações matemáticas fundamentais entre dois números informados pelo usuário.

O sistema suporta as quatro operações aritméticas principais: soma, subtração, multiplicação e divisão. Os valores são capturados a partir de campos de entrada, validados para garantir que sejam números e, em seguida, processados pela função correspondente à operação escolhida.

Também há uma verificação específica para impedir divisão por zero. Após o cálculo, o resultado é exibido dinamicamente na interface sem necessidade de recarregar a página.

## MVP

- Entrada de dois números pelo usuário
- Operações de soma, subtração, multiplicação e divisão
- Validação para garantir que os valores sejam numéricos
- Proteção contra divisão por zero
- Exibição dinâmica do resultado na interface
