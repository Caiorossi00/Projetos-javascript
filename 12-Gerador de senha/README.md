Gerador de senhas aleatórias desenvolvido em JavaScript com manipulação de DOM. A aplicação permite que o usuário defina a quantidade de caracteres desejada para a senha e, a partir desse valor, o sistema gera automaticamente uma combinação aleatória de letras e números.

O algoritmo seleciona caracteres aleatórios de uma sequência pré-definida utilizando um laço de repetição e geração de índices aleatórios. Antes da geração, o sistema valida a entrada para garantir que o tamanho da senha seja maior que seis caracteres.

Após a geração, a senha é exibida dinamicamente na interface sem necessidade de recarregar a página.

## MVP

- Definição da quantidade de caracteres da senha
- Validação para garantir tamanho mínimo da senha
- Geração aleatória de caracteres alfanuméricos
- Construção dinâmica da senha
- Exibição da senha gerada na interface
