Sistema de navegação por abas desenvolvido em JavaScript com manipulação de DOM. A aplicação permite alternar entre diferentes categorias de conteúdo (filmes, livros e jogos), exibindo dinamicamente os itens correspondentes à aba selecionada.

O sistema utiliza um objeto como fonte de dados para armazenar os itens de cada categoria e mantém o estado da aba ativa em uma variável. Ao trocar de aba, o conteúdo exibido é atualizado e o botão correspondente recebe uma indicação visual de ativo.

A renderização dos itens é feita dinamicamente através de métodos de array, garantindo uma atualização eficiente da interface sem recarregar a página.

## MVP

- Estrutura de abas para navegação entre categorias
- Armazenamento de dados em objeto organizado por categorias
- Controle de estado da aba ativa
- Renderização dinâmica dos itens na tela
- Alternância de conteúdo ao trocar de aba
- Destaque visual da aba ativa
