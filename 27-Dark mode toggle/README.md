Alternador de tema (dark mode) desenvolvido em JavaScript com manipulação de DOM. A aplicação permite ao usuário alternar entre os modos claro e escuro através de um botão, modificando dinamicamente as cores da interface.

O sistema mantém um estado que representa o tema atual e, ao ser acionado, alterna entre os valores "claro" e "escuro". Com base nesse estado, o estilo do elemento `body` é atualizado, alterando as cores de fundo e texto em tempo real.

## MVP

- Botão para alternar entre temas
- Controle de estado do tema atual
- Alternância entre modo claro e escuro
- Atualização dinâmica das cores da interface
- Manipulação de estilos diretamente no DOM
