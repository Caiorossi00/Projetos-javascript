Filtro de nomes desenvolvido em JavaScript com manipulação de DOM. A aplicação exibe uma lista de nomes e permite ao usuário realizar buscas em tempo real a partir de um campo de entrada.

O sistema utiliza um array de nomes como fonte de dados e aplica um filtro baseado no valor digitado pelo usuário. A busca é feita de forma case insensitive, garantindo correspondência independentemente de letras maiúsculas ou minúsculas. A lista exibida é atualizada dinamicamente conforme o termo de busca.

## MVP

- Lista de nomes pré-definida
- Exibição inicial de todos os nomes
- Campo de busca para filtrar resultados
- Filtro dinâmico em tempo real
- Comparação sem diferenciação entre maiúsculas e minúsculas
- Atualização automática da lista na interface
