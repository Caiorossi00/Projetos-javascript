Campo de busca com autocomplete desenvolvido em JavaScript com manipulação de DOM. A aplicação sugere nomes automaticamente conforme o usuário digita, exibindo uma lista filtrada em tempo real.

O sistema monitora o input do usuário, filtra um array de nomes com base no valor digitado e renderiza dinamicamente as sugestões correspondentes. Ao clicar em uma sugestão, o valor é preenchido no campo de busca e a lista é limpa.

A busca é case insensitive, garantindo correspondência independente de letras maiúsculas ou minúsculas.

## MVP

- Campo de busca com entrada de texto
- Filtragem dinâmica de nomes conforme digitação
- Exibição de sugestões em tempo real
- Seleção de sugestão ao clicar
- Preenchimento automático do input
- Limpeza da lista após seleção
